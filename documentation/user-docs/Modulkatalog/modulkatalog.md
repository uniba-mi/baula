# Modulkatalog


Der Modulkatalog enthält das gesamte Modulangebot für deinen Studiengang. Module sind thematische Blöcke, die du in deinen Studienverlaufsplan einplanen kannst. Module können wiederum Lehrveranstaltungen enthalten. 

Hier kannst du Module suchen und filtern:
[!ref](./suchleiste.md)

Außerdem ist es dir möglich, eine Detailansicht für jedes Modul zu öffnen, um weitere Informationen zu dem Modul zu erhalten:
[!ref](./modulDetails.md)
