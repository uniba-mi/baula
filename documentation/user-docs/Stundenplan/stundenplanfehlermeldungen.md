# Stundenplan-Fehlermeldungen erklärt

Wenn du deinen Stundenplan erstellst, kann es sein, dass ein Problem auftritt. Wenn das direkt vom System bemerkt wird, kriegst du direkt eine Fehlermeldung. Diese sind rot hinterlegt, wenn starker Handlungsbedarf besteht und gelb, wenn ein leichter Handlungsbedarf besteht.

Die möglichen Fehlermeldungen werden im Folgenden kurz erläutert:

- Die beiden Lehrveranstaltungen X & Y überschneiden sich an n Terminen!: Dies informiert dich darüber, dass es bei dir eingeplanten Lehrveranstaltungen Überschneidungen gibt. Dies kann z.B. auftreten, wenn die Lehrveranstaltungen gleichzeitig stattfinden. Durch einen Klick auf „Überschneidung auflösen“ kannst du anschließend auswählen, ob eine, oder mehrere, Lehrveranstaltungen aus deinem Stundenplan entfernt werden sollen.
- Zum Modul X wurden noch keine Lehrveranstaltungen eingeplant!: Dies gibt dir an, dass zwar ein Modul von dir in dem jeweiligen Semester eingeplant wurde, allerdings noch keine dazugehörige Lehrveranstaltung.
