Herzlich Willkommen

In diesem Bereich haben Sie die Möglichkeit, eigene Ziele in Form von ECTS Grenzen für die wichtigsten Kompetenzstandards der Lehramtsausbildung zu setzen. Die Funktionen innerhalb dieser Komponente sollen ihnen die kompetenzorientierte Auswahl von Lehrveranstaltungen in Ihrem Erziehungswissenschaftlichen Studium vereinfachen. Die nötigen Zielsetzungen können Sie jetzt direkt angeben. Alternativ können Sie auf "Abbrechen" klicken und die Ziele später über die "Einstellungen" setzen.
